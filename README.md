# CalcEngine

A simple calculator application built using C.

## 🚀 Features
- Addition
- Subtraction
- Multiplication
- Division
- Continuous operation using loop
- Error handling (division by zero)

## 🛠️ Tech Used
- C Programming Language

## ▶️ How to Run
1. Compile the code:
   gcc calculator.c -o calc

2. Run:
   ./calc
